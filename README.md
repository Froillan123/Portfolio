## Getting Started

1. Clone the repository:
```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
```
2. Install dependencies:
```sh
npm install
```
3. Start the development server:
```sh
npm run dev
```
